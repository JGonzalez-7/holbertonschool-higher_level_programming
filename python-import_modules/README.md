# Python - Import & Modules

Holberton exercises for practicing Python imports, module organization, and argument handling. Run scripts with `python3` from this directory.

## Files
- `0-add.py` — import `add` from `add_0.py` and print the sum of `1` and `2`.
- `1-calculation.py` — import arithmetic helpers from `calculator_1.py` and print the results of basic operations on `10` and `5`.
- `2-args.py` — display the argument count and list each argument with its index.
- `3-infinite_add.py` — sum an arbitrary number of integer arguments and print the total.
- `4-hidden_discovery.py` — list all names defined in the compiled module `hidden_4.pyc` (download with `curl` before running).
- `5-variable_load.py` — import `a` from `variable_load_5.py` and print its value.

## Usage
```bash
python3 0-add.py
python3 1-calculation.py
python3 2-args.py arg1 arg2
python3 3-infinite_add.py 5 10 15
curl -o hidden_4.pyc \"https://intranet.hbtn.io/files/33366/hidden_4.pyc\"
python3 4-hidden_discovery.py
python3 5-variable_load.py
```
