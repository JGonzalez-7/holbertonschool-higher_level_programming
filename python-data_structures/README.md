# Python - Data Structures: Lists, Tuples

Exercises on Python sequences: traversing lists, safely accessing elements, cloning and replacing items, working with tuples, and simple algorithms.

## Files
- `0-print_list_integer.py`: print all integers in a list, one per line.
- `1-element_at.py`: return the element at a specific index or `None` if out of range.
- `2-replace_in_list.py`: replace an element at a given index in-place when the index exists.
- `3-print_reversed_list_integer.py`: print a list of integers in reverse order.
- `4-new_in_list.py`: return a copy of a list with one element replaced at the requested index.
- `5-no_c.py`: return a new string with all `c` and `C` characters removed.
- `6-print_matrix_integer.py`: neatly print a matrix of integers, row by row.
- `7-add_tuple.py`: add two tuples element-wise, padding missing values with `0`.
- `8-multiple_returns.py`: return a tuple with the length of a string and its first character (or `None` when empty).
- `9-max_integer.py`: find the maximum integer in a list.
- `10-divisible_by_2.py`: return a list of booleans indicating which elements are even.
- `11-delete_at.py`: delete the item at a given index from a list.
- `12-switch.py`: swap the values of variables `a` and `b` without using a temporary variable.

## Usage
Run scripts directly with Python 3:
```bash
python3 <script>.py
```

## Requirements
- Python 3.8+ (Holberton baseline: Ubuntu 20.04 with Python 3.8).
- `pycodestyle` 2.8.* for style checks.
- Scripts intended to run directly should include the shebang `#!/usr/bin/python3` and be executable (`chmod +x <script>.py`).
