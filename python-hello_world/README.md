# 0x00. Python - Hello, World

Introductory Python scripts focusing on running code, formatted output, and basic string operations.

## Files
- `2-print.py`: prints `"Programming is like building a multilingual puzzle"`.
- `3-print_number.py`: prints an integer with text using proper formatting.
- `4-print_float.py`: prints a float rounded to two decimal places.
- `5-print_string.py`: repeats a string three times and shows its first nine characters.
- `6-concat.py`: concatenates `Holberton` and `School` into one greeting.
- `7-edges.py`: demonstrates slicing (`first`, `last`, and middle sections of a word).
- `8-concat_edges.py`: rebuilds a phrase by slicing and concatenating parts of a longer string.
- `9-easter_egg.py`: prints the Zen of Python.

## Running scripts
```bash
python3 <script_name>.py
```

## Style checks
```bash
pycodestyle *.py
```
