# 0x01. Python - if/else, loops, functions

Intro exercises on conditionals, loops, and simple functions in Python.

## Files
- `0-positive_or_negative.py`: pick a random integer and print whether it is positive, zero, or negative.
- `1-last_digit.py`: print the last digit of a random integer with a short description.
- `2-print_alphabet.py`: print the lowercase alphabet.
- `3-print_alphabt.py`: print the lowercase alphabet excluding `q` and `e`.
- `4-print_hexa.py`: print numbers 0–98 in decimal and hexadecimal.
- `5-print_comb2.py`: print numbers 0–99 with leading zeros, comma-separated.
- `6-print_comb3.py`: print all combinations of two different digits in ascending order.
- `7-islower.py`: function that checks if a character is lowercase.
- `8-uppercase.py`: function that prints a string in uppercase.
- `9-print_last_digit.py`: function that prints and returns the last digit of a number.
- `10-add.py`: function that adds two integers.
- `11-pow.py`: function that computes a power.
- `12-fizzbuzz.py`: print numbers 1–100 with FizzBuzz rules.

## Usage
Run scripts directly with Python 3:
```bash
python3 <script>.py
```

## Requirements
- Python 3.8+ (Holberton baseline: Ubuntu 20.04 with Python 3.8).
- `pycodestyle` 2.8.\* for style checks.
- Scripts intended to run directly should include the shebang `#!/usr/bin/python3` and be executable (`chmod +x <script>.py`).
